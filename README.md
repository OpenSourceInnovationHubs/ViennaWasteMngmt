# ViennaWasteMgmt

The ViennaWasteMgmt Application aims to explore, prototype, and demonstrate solutions for efficient, sustainable, and intelligent waste management in urban environments. It shows locations of publicly available waste containers for plastic, glass, paper and organic waste which can be visualized selectively.

## Getting started

1. Download the dataset for the waste bins form [data.gv.at](https://www.data.gv.at/datasets/8d482810-26da-4bbc-8599-6b1fd1ab8743)
2. Install the Python requirements from `requirements.txt`
3. Run `python run.py`
